# FPGA-Based Hardware Projects

This directory contains **FPGA-based hardware design projects** focusing on **real-time processing**, **RTL implementation**, and **hardware–software interfacing**.These projects demonstrate practical experience with **FPGA development workflows** and system-level hardware design.

---

## Projects Included

### Real-Time Weather Classification System
- FPGA-based implementation of weather classification logic
- RTL design derived from trained decision tree logic
- Integration of sensors and communication interfaces

### Real-Time CLAHE–Canny Edge Detection
- Hardware implementation of image processing algorithms
- Real-time processing on FPGA platform
- Performance evaluation using image quality metrics

---

## Tools and Technologies
- Verilog HDL
- Vivado / Vivado HLS
- FPGA development boards
- UART and peripheral interfacing

---

## Learning Outcomes
- RTL-based hardware design for real-time systems
- FPGA synthesis, implementation, and debugging
- Hardware interfacing and system integration
